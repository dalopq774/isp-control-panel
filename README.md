# 🛠 ISP Support Diagnostic Panel

Простая веб-панель для оперативной диагностики сетей клиентов интернет-провайдера.

---

## 📖 Описание

**ISP Support Diagnostic Panel** — экспериментальная панель управления, оптимизированная для упрощения работы операторов техподдержки провайдеров. Предоставляет быстрый доступ к основным инструментам сетевой диагностики непосредственно из браузера пользователя.

---

## ✨ Функционал

Реализованы базовые возможности диагностики:

- 🌐 Определение IP-адреса пользователя
- ⚡ Тест скорости соединения
- 📡 DNS-диагностика
- 🧭 Рекомендации по решению распространенных проблем
- 🔐 Авторизация пользователей
- 🖥 Локальное использование через браузер

---

## ⚙️ Технология

Используемые технологии:

- **PHP**
- **MySQL**
- **JavaScript**
- **HTML/CSS**
- **phpMyAdmin**

## 📦 Установка

### Требования

- Локальный веб-сервер (например, Open Server Panel)
- Поддержка PHP, MySQL, phpMyAdmin

### Шаги установки

#### 1. Клонирование репозитория

```bash
git clone https://github.com/your-repository/project-name.git
```

#### 2. Размещение файлов

Поместите файлы проекта в директорию вашего локального сервера (`domains`).

#### 3. Импорт базы данных

Импортируйте SQL-файл (`database.sql`) через phpMyAdmin.

#### 4. Настройка путей и доменов

Проверьте пути к файлам и настройте подключение к БД согласно вашей конфигурации.

## ❓ Проблемы

Ошибка "Not allowed to load local resource":

Запустите Google Chrome с параметром:

```
chrome.exe --allow-file-access-from-files --disable-web-security http://localhost/mysite.local
```
